Welcome to the English APP

English Academic Paraphrase Practice is an external app (LTI) for Canvas LMS that provides automated feedback and grading support for paraphrasing tasks.

How do I integrate it?

  • Head to your course and click on the Settings link in the sidebar.
  • Click the Apps tab and View App Configurations
  • Click + App to add Scriba
  • In Configuration Type choose By LTI2 Registration URL
  • In the Registration URL field insert: https://scriba-lti.herokuapp.com/register/

How do I use it?

Now that you have registered the external app, you can use it in several settings:

Assignment

You can use Scriba as a graded or diagnostic assignment, as follows:

  • Create an assignment in Canvas.
  • Click on Edit.
  • Input your assignment instructions.
  • Set points to 0 so that a diagnostic assignment is created.
  • In Submission type choose External tool > Scriba.
